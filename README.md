# MudaeBot

## Installation

```bash
    git clone git@github.com:GuillaumeLarue/MudaeBot.git
    cd MudaeBot
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt  
```

## Use

Change your discord IDs in the Dockerfile.
```bash
    docker compose up -d
```
