# MudaeBot

## Installation

```bash
    git clone git@github.com:GuillaumeLarue/MudaeBot.git
    cd MudaeBot
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
```

## Use

```bash
    python3 main.py -i your_discord_id -p your_discord_password
```
